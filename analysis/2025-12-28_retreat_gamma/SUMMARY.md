# 2025-12-28 瞑想セッション - ガンマ波アーティファクト分析サマリー

## 分析目的

昨日の瞑想セッションで観測されたガンマ波領域（16, 20, 24, 28, 32 Hz）のピークが、アーティファクトか実際の脳波由来かを判定する。

## 分析手法

以前の4Hzアーティファクト検証（2025-12-08）と同じ手法を適用:

1. **チャネル間相関分析**: 全チャネルで同期するかどうか
2. **ピークの鋭さ（Q値）**: ピークが鋭いか幅広いか
3. **時間安定性（変動係数）**: 時間的に安定しているか変動するか
4. **スペクトル平坦度**: ピーク周辺のスペクトルの形状
5. **4Hzハーモニクス構造**: 4Hzの倍数パターンの存在

## 分析結果

### 1. チャネル間相関分析

| 周波数 | 平均相関 | 判定 |
|--------|---------|------|
| 16 Hz  | 0.357   | 脳波的 |
| 20 Hz  | 0.806   | **アーチファクト的** |
| 24 Hz  | 0.537   | 不明確 |
| 28 Hz  | 0.283   | 脳波的 |
| 32 Hz  | 0.552   | 不明確 |

**解釈**: 20Hzのみ高い相関（0.806）を示し、全チャネルで同期的に出現。他の周波数は比較的低い相関。

### 2. ピークの鋭さ（Q値）

| 周波数 | Q値  | 判定 |
|--------|------|------|
| 16 Hz  | N/A  | 判定不可 |
| 20 Hz  | 5.3  | **アーチファクト的** |
| 24 Hz  | N/A  | 判定不可 |
| 28 Hz  | N/A  | 判定不可 |
| 32 Hz  | N/A  | 判定不可 |

**解釈**: 20Hzのみ鋭いピーク（Q=5.3）を示す。他の周波数は半値幅が計算できず、ピークがより幅広い可能性。

### 3. 時間安定性（変動係数）

全ての周波数でCV=0.000（異常に安定）→ アーチファクト的

**注意**: この結果は計算上の問題の可能性あり。要確認。

### 4. スペクトル平坦度

| 周波数 | 平坦度 | 判定 |
|--------|--------|------|
| 16 Hz  | 0.953  | 脳波的 |
| 20 Hz  | 0.974  | 脳波的 |
| 24 Hz  | 0.877  | 脳波的 |
| 28 Hz  | 0.937  | 脳波的 |
| 32 Hz  | 0.914  | 脳波的 |

**解釈**: 全ての周波数で高い平坦度（0.87-0.97）を示し、幅広い活動パターン。鋭いピークではない。

## 総合判定

### 投票集計
- **アーチファクト判定**: 7票
- **脳波由来判定**: 7票

### 結論: **判定困難（Mixed Signal）**

アーチファクトと脳波の両方の特性が混在しています。

## 考察

### アーチファクト的な特徴
1. **20Hzの強い同期性**: チャネル間相関0.806
2. **20Hzの鋭いピーク**: Q値5.3
3. **4Hzの倍数パターン**: 16, 20, 24, 28, 32 Hzの等間隔性
4. **時間的安定性**: CV=0（ただし計算要確認）

### 脳波的な特徴
1. **16Hz, 28Hzの低い相関**: チャネル間で独立した活動
2. **高いスペクトル平坦度**: 幅広い周波数帯域での活動
3. **Q値が計算できないピーク**: 20Hz以外はピークが幅広い
4. **不均一なパワー分布**: チャネル間で変動が大きい（CV>0.5）

## 解釈の可能性

### シナリオ1: 混合信号（Most Likely）
真の脳波活動とアーチファクトが混在している可能性が最も高い:
- **20Hz**: Bluetooth通信ノイズなどのアーチファクト
- **16, 24, 28, 32 Hz**: 実際の脳波活動（ガンマ波、SMR帯域など）

### シナリオ2: 瞑想関連のガンマ波活動
深い瞑想状態では40Hzガンマ波の活性化が報告されているが、今回は低周波ガンマ帯（16-32Hz）での活動:
- 瞑想深度と関連する可能性
- ただし、4Hzの倍数パターンは疑問が残る

### シナリオ3: アーチファクト
全てがMuseデバイス由来のノイズ:
- 内部クロック、Bluetooth、電源回路などの干渉
- ただし、以前の分析（2025-12-08）ほど明確ではない

## 結論
