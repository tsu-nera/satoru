## 3. SMR（Sensorimotor Rhythm）について

### 3.1 SMRとは

**SMR（Sensorimotor Rhythm）** は12-15Hzの脳波リズムで、主に運動野（中心溝付近）で観察されます。

| 特徴 | 説明 |
|:-----|:-----|
| 周波数帯域 | 12-15 Hz |
| 発生部位 | 感覚運動野（中心溝付近） |
| 増加する条件 | 身体を静止させている時、運動を抑制している時 |
| 関連する状態 | 集中、落ち着き、運動抑制 |

### 3.2 今回のデータにおけるSMR

#### 前頭部（AF7/AF8）での検出

- **主要ピーク**: 12.5 Hz（SMR帯域）
- 前頭部センサーがSMR活動を捉えている
- セッション全体を通じて持続

#### 側頭部（TP9/TP10）での検出

- **二次/三次ピーク**: 12.5 Hz, 14.25 Hz
- 8.5Hz（典型Alpha）が主だが、SMR帯域も存在
- パワーは前頭部より弱いが、検出される

### 3.3 なぜSMRが見られたか

考えられる理由：

1. **瞑想中の運動抑制**
   - 「動かないでいる」意識的な努力
   - 身体を静止させることでSMRが増加

2. **集中状態**
   - SMRは集中時に増加することが知られている
   - 瞑想中の注意集中が反映された可能性

3. **覚醒レベル**
   - 通常より「起きている」状態だった
   - 眠気が少なく、アラートな状態が持続

### 3.4 SMRの解釈

**これは必ずしも「悪い」ことではありません。**

- SMRの増加は**ニューロフィードバック訓練の目標**の一つ
- ADHD治療などでSMRを増加させる訓練が行われている
- 集中力、落ち着き、運動制御と関連

ただし、瞑想の文脈では：
- 深いリラックス状態では通常Alpha（8-10Hz）が優位
- SMR優位は「リラックスしきれていない」可能性も示唆

## 5. 結論

### 原因の特定

| 可能性 | 判定 |
|:-------|:-----|
| 接触不良 | ✗ 除外（HSI 100%） |
| 電磁干渉・ノイズ | ✗ 除外（ピーク幅が広い） |
| 前頭部のSMR/高Alpha | ✓ **最有力** |
| 脳の個人差・日差 | ✓ 可能性あり |

### 最終結論

今回の10-15Hz帯域の異常は：

1. **前頭部（AF系）で強いSMR/高Alpha（12.5Hz）が検出された**
2. **側頭部（TP系）の典型Alpha（8.5Hz）と同時に存在**
3. **スペクトログラムに両方が表示され、10-15Hz帯域が目立った**
4. **ハードウェア問題やノイズではなく、脳活動の反映**
