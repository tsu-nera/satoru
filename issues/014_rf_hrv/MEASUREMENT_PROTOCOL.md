# 共鳴周波数測定プロトコル（再測定版）

**作成日**: 2026-01-18
**目的**: 測定順序効果を排除し、個人の正確な共鳴周波数を特定する

---

## 📋 測定概要

### 測定パラメータ

- **測定回数**: 9回（各5分間）
- **呼吸レート範囲**: 3.5 - 7.0 breaths/min（0.5刻み）
- **再現性確認**: 5.0 bpmを2回測定
- **総所要時間**: 約70分（測定45分 + 休憩25分）
- **測定順序**: ランダム化済み（測定順序効果を排除）

### 前回測定の問題点と改善

| 問題点 | 改善策 |
|:------|:------|
| 測定順序効果（相関係数0.560） | ランダム化された測定順序 |
| 4.5 bpm未満の測定なし | 3.5 bpmと4.0 bpmを追加 |
| 6.5 bpmの再現性が低い（CV 19.16%） | 5.0 bpmで2回測定して再現性確認 |
| 測定環境・準備が不統一 | 標準化された測定プロトコル |

## ⚙️ 測定準備

### 1. 環境設定

- **場所**: 静かで落ち着ける部屋
- **温度**: 快適な室温（20-25℃推奨）
- **照明**: 柔らかい照明（明るすぎない）
- **音**: 外部ノイズを最小限に
- **時間帯**: 可能な限り同じ時間帯（午前中推奨）

### 2. 個人の準備

測定当日:
- ✅ カフェイン摂取を避ける（測定3時間前から）
- ✅ アルコール摂取を避ける（測定12時間前から）
- ✅ 激しい運動を避ける（測定2時間前から）
- ✅ 軽食は測定1時間前までに済ませる
- ✅ トイレを済ませておく

測定開始10分前:
- 座位で安静にする
- リラックスした状態を作る
- 呼吸を整える
- スマートフォンの通知をオフにする

### 3. 機器準備

- **Elite HRVアプリ**: 最新版にアップデート
- **心拍センサー**: フル充電、正しく装着
- **スマートフォン**: フル充電、機内モードまたはDNDモード
- **タイマー**: 休憩時間管理用

## 📝 測定手順

### 各トライアルの流れ

```
安静(10分) → 測定(5分) → 休憩(2-3分) → 次のトライアル
```

### 詳細ステップ

#### ステップ1: 測定前（各トライアルの前）

1. **姿勢を整える**:
   - 背もたれのある椅子に座る
   - 背筋を伸ばし、リラックス
   - 両足を床につける
   - 手は膝または太ももの上に

2. **心拍センサーを確認**:
   - 正しく装着されているか
   - 接続が安定しているか

3. **Elite HRVアプリを準備**:
   - 呼吸ペーシング機能を選択
   - 該当する呼吸レートを設定
   - 吸気・呼気時間を確認（上記表参照）

#### ステップ2: 測定中（5分間）

1. **呼吸ガイドに従う**:
   - アプリの視覚ガイドに合わせて呼吸
   - 吸気: ガイドの拡大に合わせて鼻から息を吸う
   - 呼気: ガイドの縮小に合わせて口から息を吐く
   - 無理のない自然な呼吸を心がける

2. **注意点**:
   - 過度に深く吸わない
   - リズムが乱れたら次のサイクルで調整
   - 体の感覚に意識を向ける
   - 思考が浮かんでも気にしない

3. **避けるべき行動**:
   - 話す、笑う
   - 体を動かす
   - スマートフォンを見る
   - 考え事に没頭する

#### ステップ3: 測定後

1. **データ保存**:
   - 測定データを確認
   - ファイル名: `YYYY-MM-DD HH-MM-SS.txt`形式で保存
   - 保存先: `issues/014_rf_hrv/data/` ディレクトリ
   - トライアル番号と呼吸レートをメモ

2. **記録**:
   以下の情報を記録（後で分析に使用）:
   - トライアル番号
   - 呼吸レート
   - 測定時刻
   - 体調・気分（1-10スケール）
   - 特記事項（あれば）

3. **休憩（2-3分）**:
   - 自然な呼吸に戻る
   - 軽くストレッチ（必要であれば）
   - 水分補給
   - 次のトライアルの準備

#### ステップ4: 全測定完了後

1. **データ確認**:
   - 9回分のファイルが保存されているか
   - ファイル名が正しいか
   - 各ファイルにデータが含まれているか

2. **分析準備**:
   - `rf_hrv_analysis.py`のtrials変数を更新
   - `measurement_protocol.txt`の内容を参照

## ⚠️ トラブルシューティング

### 測定中の問題

| 問題 | 対処法 |
|:-----|:------|
| 呼吸が苦しい | 無理せず自然な深さに調整、次のサイクルで再び合わせる |
| センサーが外れた | すぐに装着し直し、測定を継続（後で確認） |
| リズムを失った | 焦らず、次のサイクルから再び合わせる |
| 眠くなった | 目を開ける、姿勢を正す、必要なら測定を中断して休憩 |
| 集中できない | 呼吸の感覚（鼻・胸・腹）に意識を向ける |

### データ保存の問題

| 問題 | 対処法 |
|:-----|:------|
| データが保存されない | Elite HRVアプリの設定を確認、ストレージ容量確認 |
| ファイル名が不明 | すぐにトライアル番号と呼吸レートをメモ |
| データが短い | 5分未満の場合は再測定を検討 |

## 📚 参考情報

### 呼吸ペーシングのコツ

1. **最初は難しく感じても正常**:
   - 2-3サイクルで慣れてくる
   - 完璧を目指さず、リラックスを優先

2. **低い呼吸レート（3.5-4.5 bpm）**:
   - 非常にゆっくり
   - 10秒以上かけて呼吸
   - 過換気に注意

3. **高い呼吸レート（6.5-7.0 bpm）**:
   - やや速め
   - 8-9秒で1サイクル
   - 浅くなりすぎないよう注意
