## 🔬 検証する指標

### 1. **相対パワー（%）**
- Delta%, Theta%, Alpha%, Beta%, Gamma%
- 個人内標準化により日間変動を軽減

### 2. **バンド比率**
- θ/α比: 瞑想深度
- β/α比: 覚醒度
- β/θ比: 注意・集中度

### 3. **絶対パワー（dB）**
- 比較用に継続
